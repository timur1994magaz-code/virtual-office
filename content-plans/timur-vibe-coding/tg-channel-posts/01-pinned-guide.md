# 📌 ЗАКРЕП: Как установить своего AI-агента за 30 минут

> Это пошаговый гайд. Следуй ему — и у тебя будет свой AI-агент, которому можно делегировать задачи голосом через Telegram.

---

## Что ты получишь

🤖 AI-агент, который:
- Понимает голосовые сообщения
- Создаёт сайты, приложения, ботов
- Отправляет email, парсит данные
- Управляет календарём
- Работает 24/7 через Telegram

---

## Что нужно

**Вариант 1: VPS-сервер (рекомендую)**
- Любой VPS от 2 ГБ RAM (~500 ₽/мес на Hetzner/Timeweb)
- Ubuntu 22.04 или 24.04

**Вариант 2: Свой компьютер**
- Mac Mini / MacBook / Windows / Linux
- Компьютер должен быть включён (агент работает пока он работает)

---

## Шаг 1: Установка OpenClaw

OpenClaw — это платформа, на которой живёт ваш агент. Под капотом — Claude (Anthropic), самая умная модель на рынке.

**На Mac / Linux / WSL2:**
```
curl -fsSL https://openclaw.ai/install.sh | bash
```

**На Windows (PowerShell):**
```
iwr -useb https://openclaw.ai/install.ps1 | iex
```

Установщик сам:
✅ Проверит версию Node.js
✅ Установит всё что нужно
✅ Запустит мастер настройки

---

## Шаг 2: Первоначальная настройка

После установки запустится мастер. Он спросит:

1. **API-ключ для AI-модели**
   - Зайди на https://console.anthropic.com
   - Зарегистрируйся, пополни баланс ($5-20 хватит надолго)
   - Создай API-ключ → вставь в мастер

2. **Имя агента** — как хочешь его называть

3. **Демон (служба)** — скажи "да", чтобы агент работал в фоне

---

## Шаг 3: Подключение Telegram

Чтобы общаться с агентом через Telegram:

1. Открой @BotFather в Telegram
2. Отправь /newbot
3. Придумай имя и юзернейм
4. Получи токен (вида 123456:ABC-DEF...)
5. В терминале:
```
openclaw configure --section telegram
```
6. Вставь токен → готово!

Теперь пиши боту в Telegram — он отвечает. Отправляй голосовые — он понимает.

---

## Шаг 4: Подключение голосовых (Whisper)

Чтобы агент понимал голосовые сообщения:

1. Зайди на https://platform.openai.com
2. Создай API-ключ OpenAI
3. Пополни баланс ($5 хватит на тысячи сообщений)
4. Пропиши ключ в конфигурации

Теперь агент расшифровывает голосовые через Whisper и выполняет задачи.

## Шаг 5: Настройка навыков

```

```

Если всё зелёное ✅ — ваш агент готов к работе!
